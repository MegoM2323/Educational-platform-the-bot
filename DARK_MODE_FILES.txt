USAGE SUMMARY
=============

Installation/Setup:
  - No installation needed, everything is pre-configured
  - Just use the hooks and components

Basic Usage (in any component):
  import { useThemeMode } from '@/contexts/ThemeProvider';
  const { isDark, toggleTheme } = useThemeMode();

UI Component:
  import ThemeSwitcher from '@/components/ThemeSwitcher';
  <ThemeSwitcher variant="dropdown" showLabel />

Styling:
  - Tailwind dark mode: className="bg-white dark:bg-slate-900"
  - CSS variables: style={{ backgroundColor: 'hsl(var(--background))' }}

Testing:
  npm test -- src/styles/__tests__/themes.test.ts
  npm test -- src/contexts/__tests__/ThemeProvider.test.tsx
