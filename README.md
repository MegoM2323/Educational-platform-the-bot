# 🎓 THE BOT Platform

Современная образовательная платформа нового поколения с персонализированным обучением, трекингом прогресса и прямой связью между всеми участниками образовательного процесса.

## ✨ Особенности

### 🎯 Для Учеников
- 📚 Доступ к образовательным материалам и заданиям
- 📊 Трекер прогресса с геймификацией (баллы, стрики, точность)
- 💬 Чат с преподавателями и тьюторами в реальном времени
- 🔔 Уведомления о проверке домашних заданий
- 📈 Визуализация образовательного прогресса

### 👨‍👩‍👧 Для Родителей
- 📋 Отчёты о прогрессе детей
- 🤝 Связь с тьютором
- 🎯 Доступ к образовательным целям
- 💳 Управление платежами за обучение
- 🔔 Уведомления о важных событиях

### 👨‍🏫 Для Преподавателей
- 📝 Создание и публикация образовательных материалов
- ✅ Проверка домашних заданий
- 📊 Формирование отчётов по предмету
- 💬 Ответы на вопросы учеников
- 📚 Управление предметами и заданиями

### 🎓 Для Тьюторов
- 👥 Управление группой учеников
- 📄 Создание финальных отчётов
- 🤝 Координация с преподавателями
- 💬 Консультации родителей
- ➕ Создание новых учётных записей для учеников и родителей

## 🛠 Технологический стек

### Backend
- **Django 5.2+** — веб-фреймворк
- **Django REST Framework** — RESTful API
- **Django Channels** — WebSocket поддержка для реального времени
- **PostgreSQL / SQLite** — база данных (Supabase для продакшена)
- **Redis** — кэширование и очереди задач
- **Celery** — асинхронные задачи
- **Daphne** — ASGI сервер для WebSocket

### Frontend
- **React 18** — UI библиотека
- **TypeScript** — типизированный JavaScript
- **Vite** — современный сборщик
- **React Router** — маршрутизация
- **TanStack Query** — управление состоянием сервера
- **shadcn/ui + Radix UI** — компоненты интерфейса
- **Tailwind CSS** — стилизация
- **React Hook Form + Zod** — формы и валидация

### Интеграции
- **Supabase** — аутентификация и база данных
- **Telegram Bot API** — уведомления
- **YooKassa** — платежная система
- **WebSocket** — реальное время для чата

## 🚀 Быстрый старт

Предварительно: Python 3.10+, Node.js 18+.

1) Клонировать репозиторий и перейти в каталог проекта.
2) Создать виртуальное окружение и активировать: `python -m venv .venv && source .venv/bin/activate`.
3) Установить backend-зависимости: `cd backend && pip install -r requirements.txt`.
4) Применить миграции: `python manage.py migrate`.
5) Установить frontend-зависимости: `cd ../frontend && npm install`.
6) Быстрый старт: в корне `chmod +x start.sh && ./start.sh`.

После запуска:
- 🌐 Frontend: http://localhost:8080
- 🔧 Backend API: http://localhost:8000/api/
- 👨‍💼 Admin панель: http://localhost:8000/admin
